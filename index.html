Connect Supabase

Open config.js and paste your Supabase project URL and anon key. Then run schema.sql in the Supabase SQL Editor and turn off "Confirm email" in Auth settings.

  1. Create a free project at supabase.com
  2. Project Settings → API → copy Project URL and anon/public key into config.js
  3. SQL Editor → paste the contents of schema.sql → Run
  4. Authentication → Providers → Email → toggle Confirm email off
  5. Reload this page
